[اسم المشروع: AMTC Business Services Portal]
وصف قصير:
موقع ويب تعريفي متطور لشركة خدمات هندسية ولوجستية، يتميز بتصميم عصري واستجابة كاملة مع جميع الشاشات، مدعوم بتحريكات بصرية تفاعلية.

 التقنيات المستخدمة (Tech Stack)
HTML5 & CSS3: لبناء الهيكل وتصميم الواجهة.

Bootstrap 5: لضمان استجابة الموقع (Responsiveness) ونظام الشبكة (Grid System).

GSAP (GreenSock Animation Platform): لعمل تحريكات التمرير (Scroll Animations) والظهور التدريجي.

Google Fonts: لتحسين طباعة النصوص وتجربة القراءة.

  المميزات (Features)

Hero Section: واجهة جذابة مع خلفية بصرية تعكس هوية الشركة.

Interactive Cards: استخدام ثلاثة أنماط من البطاقات (Light, Glass, Dark) لعرض الخدمات.

Project Gallery: معرض صور بتأثيرات حركية عند تمرير الفأرة (Hover Effects).

ScrollTrigger: تحريك العناصر تلقائياً عند التمرير لأسفل لزيادة تفاعل المستخدم.

Video Integration: قسم مخصص لعرض نظرة عامة عن الشركة عبر الفيديو.
